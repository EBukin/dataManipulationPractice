UN Comtrade data

  • Data availability
    • Trade availability visualization tool https://comtrade.un.org/data/da
    • API interface https://comtrade.un.org/data/Doc/API#DataAvailabilityRequests
  • Regular download https://comtrade.un.org/data/
  • Bulk download
    • Web based https://comtrade.un.org/data/bulk
    • API https://comtrade.un.org/data/doc/api/bulk/
    • Token authentication page
  • Data visualization options https://comtrade.un.org/labs/

  • WITS

  • Trade data time series http://stat.wto.org/;
